# Conky
Config files for conky Monitor

## Required dependencies
JQ
Lm-sensors
curl
lua

## Fonts
Roboto
Metropolis
Arimo Nerd Font (or another Nerd Font)

## Scripts 
+ blue for currency values
+ weather for climate info
Connect to https://openweathermap.org and get city id and api key
